Object Oriented Programming (OOP) refers to using self-contained pieces of code to develop applications. We call these self-contained pieces of code objects, better known as Classes in most OOP programming languages and Functions in JavaScript. In JavaScript, almost "everything" is an object: Booleans, numbers, and strings can be objects, and dates, maths, regular expressions, arrays, and functions are always objects. All values, except primitive values, are objects.

Constructor Function

What is it?

A constructor is a function that can be invoked using the new keyword to create new objects. Constructor functions are used as a blueprint for creating many of the same type objects, and add properties.

Code Preview

function Pizza(pizzaSize, toppings, pizzaQuantity) {
                  this.pizzaSize = pizzaSize;
                  this.pizzaToppings = [];
                  this.pizzaQuantity = pizzaQuantity
                }

Properties

What is it?

If a Constructor is the blueprint for the house, then one of the properties is the plumbing. A property is a specification of the Constructor Function that is made up of a key and value

Prototype: Toppings Price Calculator

What's it doing?

When toppings are selected they are stored into an empty array and hold a value of 1. The Toppings Price Calculator uses the .length property and returns the number items selected into a combined value.
