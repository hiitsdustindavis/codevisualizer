Object Oriented Programming (OOP) refers to using self-contained pieces of code to develop applications. We call these self-contained pieces of code objects, better known as Classes in most OOP programming languages and Functions in JavaScript. In JavaScript, almost "everything" is an object: Booleans, numbers, and strings can be objects, and dates, maths, regular expressions, arrays, and functions are always objects. All values, except primitive values, are objects.

Constructor Function

What is it?

A constructor is a function that can be invoked using the new keyword to create new objects. A function is a block of code designed to perforam a particular task. Constructor functions are used as a blueprint for creating many of the same type objects, and add properties.

Prototype

What is it?

Prototypes are objects from which other objects inherit. Prototypes store methods to be shared by all objects of the same constructor. For example:Pizza.prototype.sizePriceCalc is shared by all Pizza objects and is used in calculating the total price.

Variables:

What is it?

JavaScript variables are containers for storing data values. Variables are written as "var" in Javascript.
A variable declared outside a function becomes GLOBAL. A global variable has global scope: All scripts and functions on a web page can access it.
Local variables are only recognized inside their functions, so variables with the same name can be used in different functions. Local variables are created when a function starts, and deleted when the function is completed.
